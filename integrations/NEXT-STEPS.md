# Next Steps — Exactly What To Do

## Step 1: Get an npm account (5 min)

Go to https://www.npmjs.com/signup — create an account. Then:

```bash
npm login
```

## Step 2: Get a GitHub repo up (2 min)

```bash
cd /home/erik/Documents/MCP_Connector_App
gh auth login     # if not already
gh repo create kondi --public --source . --push
```

## Step 3: Update the repo URL in all packages (1 min)

Once you know your GitHub username, run:

```bash
cd mcp-connect-mvp
find integrations/ -name "package.json" -exec sed -i 's|yourusername|YOUR_ACTUAL_USERNAME|g' {} +
find integrations/ -name "pyproject.toml" -exec sed -i 's|yourusername|YOUR_ACTUAL_USERNAME|g' {} +
```

## Step 4: Publish (1 min)

```bash
cd /home/erik/Documents/MCP_Connector_App/mcp-connect-mvp
./integrations/publish-all.sh
```

## Step 5: Post the content

Here's where each piece goes, in priority order:

| What | Where | How | When |
|------|-------|-----|------|
| **Show HN post** | https://news.ycombinator.com/submit | Paste title + URL from `content/CONTENT-PLAN.md` Post 1 | Tuesday-Thursday, 8-10am ET |
| **Showcase posts** (rent-vs-buy, CS degree, remote work) | **LinkedIn** — https://www.linkedin.com/feed/ → "Start a post" | Paste from the showcase .md files. Add the demo GIF if you have one. | One every 2-3 days |
| **Same showcases** | **Twitter/X** — https://x.com/compose/post | Paste the first 2-3 paragraphs as a thread. First tweet is the hook line. | Same day as LinkedIn |
| **MCP server** | Smithery.ai — https://smithery.ai | Submit `kondi-council-mcp` package | After npm publish |
| **Reddit comments** | Search https://www.reddit.com/r/LocalLLaMA for "multi-agent" or "code review" threads | Paste a relevant snippet from `content/reddit-comments.md` into an existing thread | When you find a relevant thread |
| **n8n forum** | https://community.n8n.io → New Topic → "Share a workflow" | Paste from `.generated-posts/n8n-forum.md` (generated by promote.sh) | After npm publish |

## Scripts reference

```bash
./integrations/build-all.sh              # Build all 8 packages
./integrations/build-all.sh --bump patch # Bump version + build
./integrations/publish-all.sh            # Build + publish to npm/PyPI
./integrations/promote.sh               # Full launch: GitHub, publish, PRs, posts, browser tabs
./integrations/promote.sh --step posts   # Just generate post content
./integrations/promote.sh --step announce # Open browser tabs to post
```
